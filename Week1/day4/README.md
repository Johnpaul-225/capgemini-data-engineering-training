#  SQL Practice – Functions, Regex & Data Handling

##  Objective
This project focuses on practicing core SQL concepts including:
- Regular Expressions (Regex)
- Numeric Functions
- NULL Handling

The goal is to understand how to clean, transform, and extract meaningful insights from data.

---

##  Problem Summary

- Work with structured tables containing text, numbers, and null values  
- Apply regex to extract patterns from strings  
- Use numeric functions for calculations  
- Handle NULL values using built-in SQL functions  
- Solve real-world data transformation scenarios  

---

##  Modules Covered

### 🔹 1. Regex Practice

- Extract numbers from strings (start, end, middle)  
- Extract characters (first, last, patterns)  
- Extract email components (username, domain, extension)  
- Extract phone country codes  
- Extract patterns from mixed values  

---

### 🔹 2. Number Functions

- Perform mathematical operations  
- Apply rounding techniques  
- Generate random values  
- Compare numeric values  

---

### 🔹 3. NULL Handling

- Identify NULL values  
- Replace NULL values  
- Handle missing data in calculations  
- Use conditional NULL logic  

---

## ⚙️ Approach

- Created sample tables with realistic data  
- Inserted test data with mixed formats and NULL values  
- Applied SQL queries step-by-step  
- Used built-in functions to clean and transform data  
- Solved practice questions for each concept

##  Key SQL Concepts Used

### 🔹 Regex Functions
- Pattern matching  
- Extracting substrings  
- Working with mixed data  

---

### 🔹 Numeric Functions
- ABS → absolute value  
- ROUND → rounding values  
- CEIL / FLOOR → upper/lower rounding  
- MOD → remainder  
- POWER / SQRT → calculations  
- GREATEST / LEAST → comparisons  

---

### 🔹 NULL Functions
- IS NULL / IS NOT NULL → filtering  
- ISNULL() → replace NULL  
- COALESCE() → first non-null value  
- NULLIF() → conditional NULL
