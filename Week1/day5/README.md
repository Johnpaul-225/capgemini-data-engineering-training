#  SQL Practice – CASE WHEN & Window Functions

##  Objective
Practice advanced SQL concepts using real-world scenarios.  
Focus on applying business logic using CASE WHEN and performing analysis using window functions.

---

##  Problem Summary

- Analyze customer usage data  
- Apply multiple business rules using CASE WHEN  
- Classify customers based on behavior  
- Identify patterns in usage (data, calls, billing)  
- Perform ranking using window functions  

---

##  Approach

- Created CustomerUsage table with sample telecom data  
- Inserted records with different usage patterns  
- Applied CASE WHEN conditions step-by-step  
- Used window functions for ranking and analysis  
- Generated insights from the results  

---

##  Key Concepts Used

### 🔹 CASE WHEN
- Conditional logic  
- Nested conditions  
- Business rule implementation  

### 🔹 Window Functions
- ROW_NUMBER()  
- RANK()  
- DENSE_RANK()
